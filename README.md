# Item Inspect Menu
<p align="left"> <img height="350px" src="https://staticdelivery.nexusmods.com/mods/130/images/97721/97721-1780021252-1627532296.png"> </p>

## Overview
- Adds a Fallout 4 inspired item inspect menu.
- Inspect any item while in the Inventory, Container, or Barter menus.
- Optionally display the Inspect Menu after picking up unique/quest items for the first time.
- Display detailed item stats and equipment comparisons via optional Item Cards support.

## Controls
### Keyboard
- Inspect: T
- Rotate: Click + Drag
- Zoom: Mouse Wheel

### Controller
- Inspect (Inventory): RB + Y
- Inspect (Container/Barter): Y
- Rotate: R-Stick
- Zoom: LT/RT

## Configuration
Configurable via [MCM Extender](https://www.nexusmods.com/newvegas/mods/93642):
- Set inspect keybind.
- Set rotation/zoom sensitivity.
- Toggle camera type (Orthographic/Perspective)
- Display item name/description.
- Inspect unique/quest on first pickup.
